# 发票整理记账助手

一个基于 Python + Flask 的微信风格聊天界面发票整理记账网页应用。上传发票、小票、支付截图或外卖实物照片，AI 自动识别内容并按规范重命名打包。

## 功能特性

- 微信聊天风格界面，操作直观
- 支持 PDF、JPG、JPEG、PNG 格式文件上传
- 一次最多上传 10 个文件，单文件不超过 10MB
- 调用 Agnes AI（agnes-2.0-flash 模型）自动识别文件内容
- 智能判断发票、小票、支付截图、外卖实物照片四种类型
- 自动提取日期、消费类型（午餐/晚餐）、金额
- 以支付截图信息为基准，统一重命名所有文件
- 自动打包为 ZIP 文件下载

## 环境要求

- Python 3.9+
- pip

## 安装与运行

### 1. 克隆或下载代码

```bash
# 进入项目目录
cd invoice-app
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 配置 API 密钥

打开 `main.py`，找到第 29 行：

```python
AGNES_API_KEY = "请替换成你的Agnes API密钥"
```

将值替换为你的 Agnes AI API 密钥。如果你还没有密钥，请前往 [Agnes AI](https://apihub.agnes-ai.com) 注册获取。

### 4. 启动应用

```bash
python main.py
```

启动成功后，终端会显示访问地址（默认 `http://127.0.0.1:5000`，如果端口被占用会自动使用下一个可用端口）。

### 5. 打开浏览器

在浏览器中打开 `http://127.0.0.1:5000`，即可开始使用。

## 文件命名规则

| 文件类型 | 命名格式 |
|---------|---------|
| 支付截图 | X月X日-午餐/晚餐-XX元-支付截图{序号}.扩展名 |
| 发票 | X月X日-午餐/晚餐-XX元-发票{序号}.扩展名 |
| 小票 | X月X日-午餐/晚餐-XX元-小票{序号}.扩展名 |
| 外卖实物照片 | X月X日-午餐/晚餐-XX元{序号}.扩展名 |

### ZIP 文件夹命名

格式：`X月X日-午餐/晚餐-X人-XX元.zip`

示例：`7月15日-午餐-3人-128元.zip`

## 常见问题

**Q: 提示"请先在 main.py 中配置 AGNES_API_KEY"？**

A: 请按照上方"配置 API 密钥"的步骤设置你的 API 密钥。

**Q: AI 识别失败怎么办？**

A: 检查以下事项：
- API 密钥是否正确
- 网络连接是否正常
- 图片是否清晰可辨
- 文件是否超过大小限制

**Q: 端口被占用？**

A: 应用会自动尝试下一个可用端口，无需手动修改。

**Q: 上传文件有限制吗？**

A: 一次最多 10 个文件，单个文件不超过 10MB，支持 PDF、JPG、JPEG、PNG 格式。
